Гости по коду, без всякого сервера

Полка и профиль хранятся прямо в браузере (localStorage), хостинг — на GitHub Pages, бесплатно. Через карандашик ✎ у профиля можно скопировать код полки и отправить другу. Друг вставляет его в «+ полка друга» и видит вашу библиотеку. Код длинный, потому что сервера нет (данные внутри кода); с бесплатным Supabase он позже станет коротким, а вид не изменится.
